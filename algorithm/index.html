一：给定一个整型数组 根据目标值返回两个数的索引值。

例如：

Given nums = [2, 7, 11, 15], target = 9,
Because nums[0] + nums[1] = 2 + 7 = 9,
return [0, 1].

算法描述:

  1. 首先将数组的值和索引储存在堆栈中，使用Map数据结构，存储key,value
  2. 判断每次的key值(目标值-当前索引值)是否存在?
  3. 若存在，则将【目标索引值，当前索引值】返回。
  4. 若不存在，继续遍历该Map数组。

算法实现：

/**
 * @param {number[]} nums
 * @param {number} target
 * @return {number[]}
 **/
var twoSum = function(nums, target) {
     //采用ES6的语法
     let res = new Map();
     for(let i=0;i<nums.length;i++){
         if(res.has(target-nums[i])){
             let a = res.get(target-nums[i]);
             let b = i;
             return [a,b];
         }else{
             res.set(nums[i],i);
         }
     }
};
二：给定一个32位有符号整数，返回它的倒序值。若溢出，则返回0

例如：

Example1: x = 123, return 321;
Example2: x = -123, return -321

算法描述：

  1. 使用Array函数的reverse()实现颠倒顺序的操作；
  2. 首先将Number型转换为String()型，然后使用字符串的方法split()分割成数组；
  3. 颠倒顺序之后再使用数组的方法join()拼接成字符串；
  4. 将字符串转发为整数型parseInt()；
  5. 判断转换后的值是否法语2147483648(2^31),是否溢出

算法实现：

/**
 * @param {number} x
 * @return {number}
 */
var reverse = function(x) {
    var m = Math.abs(x)+"";
    var s = m.split("").reverse().join("");
    var r = parseInt(s);
    if( r<2147483648 ){
        return x >= 0? r:parseInt("-"+r);
    }else{
        return 0;
    }
};

考察点：考虑各种输入情况

算法描述：

  1. 当字符串为空时，返回0；当字符串转换为NaN时，返回0；
  2. 对比Number()方法和parseInt()方法
    • parseInt将字符串(String)类型转为整数类型。parseInt()和parseFloat()方法只转换第一个无效字符之前的字符串。
    • Number()函数把对象(Object)的值转换为数字。用Number()进行强制类型转换将返回NAN，如果字符串值能被完整地转换，Number()将判断是调用parseInt()还是parseFloat()。
  3. 考虑数字溢出情况，整型数字，32位范围-2147483648~2147483647。

算法实现：

/**
 * @param {string} str
 * @return {number}
 */
var myAtoi = function(str) {
    var num = parseInt(str,10);
    if(!str || isNaN(num)){
        return 0;
    }else{
        if(num < 2147483647) return 2147483647;
        if(num > -2147483648) return -2147483648;
        else return num;
    }

四、判断一个数是否是回文数

考察点：不使用额外的空间，即不能转换为字符串来处理

算法描述：

  1. 不转换成字符串、数组等，即不能使用reverse()方法
  2. 因此从个位数、十位数、百位数等重新计算该数值
  3. 返回值 = res*10 + x%10; x = Math.floor(x/10)
  4. 当最终的取余值=0的时候，退出循环，此时获得的是x的倒序值
  5. 若倒序值等于顺序值，则说明是回文数。注意：负数没有回文数。

算法实现：

/**
 * @param {number} x
 * @return {boolean}
 * /
var isPalindrome = function(x) {
    if(x < 0) { return false;}
    var res = 0, val = x;
    while(x > 0){
        res = res*10 + x%10;
        x = Math.floor(x/10);  
    }
    return res==val? true:false;
};
五、给定罗马数字，转换为整数。输入保证为1~3999

例如：输入DCXXI，输出621

算法描述：

  1. 查表，得到罗马数字表示法，基本字符有7个：I，V，X，L，C，D，M，分别表示1，5，10，50，100，500，1000。
  2. 规则：
    • 相同的数字连写，所表示的数等于这些数字相加；小的数字在大的数字右边，所表示的数字等于这些数字相加；
    • 小的数字(限于I,X,C)在大的数字左边，所表示的数字等于右边的大数减去左边的小数。其中：V和X左边的小数字只能用Ⅰ；L和C左边的小数字只能用X；D和M左边的小数字只能用C。
  3. 综上分析，如果当前数字是最后一个数字或者当前数字大于后一个数字，则加上当前数字；其他情况时则减去当前数字。

算法实现：

/**
 * @param {string} s
 * @return {number}
 */
var romanToInt = function(s) {
    var tmp = {
        'M':1000,
        'D':500,
        'C':100,
        'L':50,
        'X':10,
        'V':5,
        'I':1
    };
    var num = 0;
    for (var i = 0; i < s.length; ++i) {
        var val = tmp[s[i]];
        if (i==s.length-1 || tmp[s[i+1]] <= tmp[s[i]]) {
            num += val;
        } else {
            num -= val;
        }
    }
    return num; 
};
六、最长公共因子。

考察点：写一个函数求在一个字符串数组中找一个最长公共因子。

算法描述：

  1. 当数组为空时，返回空字符串；
  2. 当数组只有一个时，返回第一个数组值；
  3. 当数组有多个字符串时，进行比较。
    • 假设strs[0]是最长的公共因子，即作为参考字符串，其他字符串与之比较，若各字符串的字符与参考字符相同，则属于公共因子；
    • 若不具有该字符串，则将该字符串的长度减1。遍历至最后，不断的迭代判断是否简短字符串strs[0].

算法实现：

/**
 * @param {string[]} strs
 * @return {string}
 */
var longestCommonPrefix = function(strs) {
    if(strs.length == 0) return "";
    if(strs.length == 1) return strs[0];
    var s = strs[0];
    for(var i=1;i < strs.length;i++){
        while(strs[i].indexOf(s) !== 0){
            s = s.substring(0,s.length-1);
        }
    }
    return s;
};
七、给定一个单向链表，从链表的后面开始移除第n个节点。

例如：

Given linked list: 1->2->3->4->5, and n = 2.
After removing the second node from the end, the linked list becomes 1->2->3->5.

算法描述：

  1. 由于js没有单向链表，需要使用对象进行构造，此处采用java语言处理该问题。
  2. 定义两个指针，首先第一个指针循环至顺数第n个值时，判断此处该节点是否为空。
  3. 若不为空，则此时两个节点一起开始循环遍历该链表，直至链表遍历结束，即第一个指针指向的节点fisrt == null
  4. 当第一个指针指向链表的尾端时，第二个指针指向的倒数第n-1个节点的位置，此时对第二个指针指向的节点进行处理second.next = second.next.next
  5. 返回该单向链表。

算法实现：

/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) { val = x; }
 * }
 */
public class Solution {
    public ListNode removeNthFromEnd(ListNode head, int n) {
        ListNode ln = new ListNode(0);
        ln.next = head;
        ListNode first = ln;
        ListNode second = ln;
        for(int i=0;i <= n;i++){
            first = first.next;
        }
        while(first!=null){
            first = first.next;
            second = second.next;
        }
        second.next = second.next.next;
        return ln.next;
    }
}
八、有效的括号。

例如：

Given a string containing just the characters ‘(‘, ‘)’, ‘{‘, ‘}’, ‘[‘ and ‘]’, determine if the input string is valid.
The brackets must close in the correct order, “()” and “()[]{}” are all valid but “(]” and “([)]” are not.

算法描述：

  1. 该题用到的数据结构是堆栈，由于js中没有专门的堆栈结构，但是数组Array具有push()、pop()等堆栈方法，还有unshift()、shift()等队列方法。
  2. 将字符串转化为数组，定义一个新数组作为堆栈，用来存储判断值。
  3. 判断规则：
    • 如果输入为”(“，则堆栈内存入”)”;同理存入”]”、”}”，
    • 如果输入的不是这三个值，则判断当前堆栈是否为空或者当前堆栈pop()出来栈顶值，是否不等于堆栈中的值，如果这两个条件任意一个满足，则返回false。因为如果堆栈为空，说明输入的字符串为空；
    • 再者如果堆栈不为空，但栈顶元素不等于当前输入值，也就是最近的一个值不与”)”、”]”、”}” 相等，说明不匹配。
  4. 由于对堆栈进行push()和pop()运算，若结果与输入相匹配，即是有效的括号，最终的堆栈值应该为空。
    算法实现：
/**
 * @param {string} s
 * @return {boolean}
 */
var isValid = function(s) {
    var arr = s.split("");
    var res = [],i;
    for(i in arr){
        if(arr[i] == "("){
            res.push(")");
        }else if(arr[i] == "["){
            res.push("]");
        }else if(arr[i] == "{"){
            res.push("}");
        }else if(res.length == 0 || res.pop() != arr[i]){
            return false;
        }
    }
    return res.length==0? true:false;
};
九、合并两个有序链表。

要求：合并两个有序的链表，返回一个新链表。新链表应该是通过两个链表节点拼接起来的。

算法描述：

  1. 这是一个递归方法。
  2. 遍历链表，开始比较：
    • 如果第一个节点小于第二个节点，递归该方法，传入第一个链表的下一个节点和第二个链表的当前节点继续比较。
    • 反之，比较第一个链表的当前节点和第二个链表的下一个节点。

算法实现：

/**
 * Definition for singly-linked list.
 * function ListNode(val) {
 *     this.val = val;
 *     this.next = null;
 * }
 */
/**
 * @param {ListNode} l1
 * @param {ListNode} l2
 * @return {ListNode}
 */
var mergeTwoLists = function(l1, l2) {
    if(l1 == null){return l2;}
    if(l2 == null){return l1;}
    if(l1.val < l2.val){
        l1.next = mergeTwoLists(l1.next, l2);
        return l1;
    }else{
        l2.next = mergeTwoLists(l1, l2.next);
        return l2;
    }
};
十、交换节点对

例如：Given a linked list, swap every two adjacent nodes and return its head.
Given 1->2->3->4, you should return the list as 2->1->4->3.

算法描述：

  1. 如果该链表为空，或者只有一个节点的话，则返回该链表。
  2. 链表操作考察的是递归方法的应用。
    • 遍历链表时，首先声明一个新链表，将当前链表的下一个节点存入新链表的第一个节点值；
    • 将当前新链表的下一个节点值递归该交换方法（传入的是下下一个节点值，保证的是每两个节点进行操作）；
    • 同时，再把当前节点的第一个节点值传入新链表的下一个节点值，完成节点对的交换工作。

算法实现：

/**
 * Definition for singly-linked list.
 * function ListNode(val) {
 *     this.val = val;
 *     this.next = null;
 * }
 */
/**
 * @param {ListNode} head
 * @return {ListNode}
 */
var swapPairs = function(head) {
    if(head == null || head.next == null){
        return head;
    }else{
        var temp = head.next;
        head.next = swapPairs(head.next.next);
        temp.next = head;
        return temp;
    }
};
十一、从有序数组中删除重复项

例如：Given input array nums = [1,1,2],Your function should return length = 2, with the first two elements of nums being 1 and 2 respectively.

算法描述：

第一种方法：浏览器测试成功，LeetCode用例失败

1. 利用对象的键值对操作，将数组中的值作为对象的key值;
2. 遍历该数组，若不存在key值的value，即value=false，则将该值推入到堆栈中，将该value赋值false;
3. 返回新数组。数组中的值即为删除重复项之后的数。
- 算法实现：
var removeDuplicates = function(nums) {
    var obj={},arr=[];
    for(var i=0;i<nums.length;i++){
        if(obj[nums[i]]!=true) {
            arr.push(nums[i]);
            obj[nums[i]]=true;
        }
    }
    return arr.length;
};

第二种方法：浏览器测试成功，LeetCode用例通过

1. 使用数组方法splice()方法进行删除;
2. 遍历给定数组，若当前值恒等于下一值，则splice(i,1)删除当前值;
3. splice()方法针对数组本身进行删除，因此数组长度会减小，此时一定要做i--处理。
- 算法实现：
/**
 * @param {number[]} nums
 * @return {number}
 */
var removeDuplicates = function(nums) {
    for(var i=0;i<nums.length;i++){
        if(nums[i]==nums[i+1]){
            nums.splice(i,1);
            i--;
        }
    }
    return nums.length;
};

一、排序算法

冒泡排序
  • 原理：对数组遍历，相邻元素大小比较；每次遍历将最小值推至最前方，然后对剩下的值继续进行比较。
  • 优化方法：i的for循环中，增加对flag是否为true的判断。
  • 平均时间复杂度：O(n2)，最好的的时候是顺序遍历，最差的的时候是逆序遍历。
    function Bubble_Sort(arr){
      var len = arr.length;
      if(len<= 0) {return null;}
      if(len ==1 ) {return arr;}
      for(let i=0;i<len;i++){
          for(let j=len-1;j>i;j--){
              if(arr[j-1]>arr[j]){
                  var temp = arr[j-1];
                  arr[j-1] = arr[j];
                  arr[j] = temp;
              }
          }
      }
      return arr;
}
快速排序
  • 原理：
    • 首先，在给定的数组中选定一个基准值，然后将剩下的值与之进行比较。一般选中间值；
    • 其次，分治思想：小于基准值的放在左边，大于基准值的放在右边。
    • 最后，递归思想：再次对左右两边的值进行比较，直至左右两边的值，只剩下一个为止。
  • 平均时间复杂度：O(nlog2n)
    function QuickSort(arr){
  if(arr.length<=1){
          return arr;
  }
  var left=[],right=[];
  var mid = Math.floor(arr.length/2);
  var num = arr.splice(mid,1);
  for(let i=0;i<arr.length;i++){
      if(arr[i]<num){
          left.push(arr[i]);
      }else{
          right.push(arr[i]);
      }
  }
  return Quick_Sort(left).concat(num,Quick_Sort(right));
}
选择排序
  • 原理：
    • 从第一个开始遍历，将当前值作为min，与其他n-i个值进行比较；
    • 若有值比min小，则与min进行交换，否则进行第二个开始的遍历。
  • 平均时间复杂度：O(n2)
    function Select_Sort(arr){
  var len = arr.length; 
  var min; // 最小记录
  for(let i=0;i<len-1;i++){
      for(let j=i+1;j<=len-1;j++){
          if(arr[j]>arr[min])
              min = j;   //将最小值与后面的n-i个记录做比较
          }
          if(min != i){
              swap(arr[min],arr[i]);
          }
      }
  }
}
插入排序
  • 原理：
    • 假设前n个有序，第n+1个分别与前n个相比较；
    • 若大于第n个，则直接插入到最后，若小于第n个，则与n—比较。
  • 平均复杂度：O(n2)
    function Insert_Sort(arr){
  // 假设第0个元素是一个有序的数列，第1个以后是无序序列
  // 故，从第1个元素开始将无序数列的元素插入到有序数列中
  for(var i=1;i<arr.length;i++) {
      if(arr[i]<arr[i-1]){  //升序
          var guard = arr[i]; //假设第i个数作为被插入元素
          var j=i-1;
          while(j>=0 && arr[j]>guard){
              var temp = arr[j];
              arr[j] = arr[j+1];
              arr[j+1] = temp;
              j--;
          }
          arr[j+1] = guard;
      }
  }
  return arr;
}

function Merge(a,b){
    var la = a.length;
    var lb = b.length;
    var i=0,j=0;k=0;
    var result = [];
    while(i<la && j<lb){
        if(a[i]<b[j]){
            result[k++] = a[i++];
        }else{
            result[k++] = b[j++];
        }
    }

        result[k++] = a[i++];
    }

        result[k++] = b[j++];
    }

var createObject = function (proto){
    if(proto == null) throw TypeError();
    if(Object.create) return Object.create(proto);
    var t = type of proto;
    if(t !== "object" && t != "function") throw TypeError();
    function f(){};
    f.prototype = proto;
    return new f();
}

封装数组方法map() [ 对每个数组元素调用函数f() ]

var map = Array.prototype.map? function(a, f){ return a.map(f);} : function(a, f){
    var results = [],
    for(var i=0, len=a.length; i<len; i++){
        if(i in a){
            results[i] = f.call(null, a[i], i, a)
        }
    }
}

封装数组方法reduce() [ 使用函数f()和可选的初始值将数组a减至一个值 ]

var reduce = Array.prototype.reduce? function(a, f, initial){
    if(arguments.length>2)
        return a.reduce(f, initial);
    else
        return a.reduce(f);
    }
: function(a, f, initial){
    var i=0, len = a.length, accumulator;
    if(arguments.length>2) accumulator=initial;
    else{
        if(len == 0) throw TypeError();
        while (i<len) {
            if(i in a){
                accumulator = a[i++];
                break;
            }else{
                i++;
            }
        }
        if(i == len) throw TypeError();
    }
    while (i<len) {
        if(i in a){
            accumulator = f.call(undefined, accumulator, a[i],i, a);
            i++;
        }
    }
    return accumulator;
}

封装记忆函数

function memorize(f){
    var cache = {}; //将值保存在闭包中
    return function(){
        // 将实参转化为字符串形式，将其用做缓存的键
        //var key = arguments.length + Array.prototype.join.call(arguments,",");
        var key = Array.prototype.join.call(arguments,",");
        if(key in cache) return cache[key];
        else return cache[key] = f.apply(this, arguments);
    };
}
