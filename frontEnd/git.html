git 常用命令

  • git status // 查看分支状态，查看文件状态

  • git add . // 把要提交的所有修改放到暂存区（Stage）

  • git commit -m ‘add new info’ //可以一次性把暂存区的所有修改提交到分支

  • git pull –rebase 分支 // 解决冲突的过程

  • git push origin [自己的分支名称]:refs/for/[自己的分支名称]%r=评审人邮

  • git checkout -b dev // 创建dev分支，然后切换到dev分支

    等价于

    • git branch dev //创建分支
  • git checkout dev //切换分支

  • git branch //查看分支状态

  • git log // 查看历史记录

  • git merge dev 把dev分支的工作成果合并到master分支上
