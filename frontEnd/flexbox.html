Flex

1.Flex布局

任何一个容器都可以指定为Flex布局，行内元素也可以使用Flex布局，Webkit内核浏览器，必须加上-webkit前缀。

.box{
  display: -webkit-flex; /* Safari */
  display: flex;
}

2.基本概念

  • 采用Flex布局的元素，成为Flex容器，其所有子元素自动成为容器成员，成为flex item，简称项目。
  • 容器默认两根轴：水平的主轴(main axis)和垂直的交叉轴(cross axis)。项目默认沿主轴排列。

3.容器属性

  • display:flex;
  • flex-direction: row|row-reverse|column|column-reverse;
  • flex-wrap:nowrap|wrap|wrap-reverse;
  • flex-flow:<’flex-direction’>||<’flex-wrap’> // 同时定义主轴和交叉轴;
  • justify-content:flex-start|flex-end|center|space-between|space-around;
  • align-items:flex-start|flex-end|center|stretch|baseline//交叉轴
  • align-content:flex-start|flex-end|center|stretch|space-between|space-around

4.项目属性

  • order: // 当前item的顺序
  • flex-grow: //当前item的增长倍数
  • flex-shrink: //当前item的收缩倍数
  • flex-basis: | auto; / default auto /
  • flex: none | [ <’flex-grow’> <’flex-shrink’>? || <’flex-basis’>
  • align-self: auto | flex-start | flex-end | center | baseline | stretch; // 当前item的位置

5.兼容性

  • new -> display:flex
  • tweener -> display:flexbox(IE10)
  • old -> display:box
