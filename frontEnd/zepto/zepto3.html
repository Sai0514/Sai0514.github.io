zepto之事件

createProxy 创建代理对象

zepto中，事件触发的时候，返回给我们的event都不是原生的event对象，都是代理对象。代理对象的创建方法。

function createProxy(event){
    var key, proxy = {originalEvent: event}
    for(key in event) {
        if(!ignoreProperties.test(key) && event[key] != undefined)
            proxy[key] = [key]
    }
    // compatible 修正event对象
    return compatible(proxy, event)
}

常用的是on,off,trigger方法，on和off主要是调用了add/remove方法，因此，接下来主要分析这两个方法。思想很重要~

  • add方法先做了event对象的修正，然后对每个事件进行处理，最后调用element.addEventListener(realEvent(handler.e), handler.proxy, eventCapture(handler, capture)) 添加事件
  • remove方法同理，最后执行removeEventListener移除事件
